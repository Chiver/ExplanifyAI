# ExplanifyAI
ExplanifyAI's Official Release Repository

## To Install 
* Down load DMG 
* Drag ExplanifyAI file out to a directory (e.g. ~/Desktop) 
* Open ExplanifyAI, grant permission
* Quit ExplanifyAI 
* Re-open Explanify AI
